## Installation

```
npm i && npm start
```

### Mac OS

You need to point the React app towards the appropriate URL. You can edit this in the `.env` file in the root directory.

```
REACT_APP_API_URL=http://<laravel url here>/api/
```

### Win OS

You need to point the React app towards the appropriate URL. You can edit this in the `.env` file in the root directory.

```
REACT_APP_API_URL=http://localhost:8000/api/
```
